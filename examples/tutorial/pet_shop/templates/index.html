This is the sparse frontend for a basic pet shop API.

To see the automatically generated documentation, look here.

To see the Open API 3.0 source, look here.


To actually test the API, you can use curl. Run:

curl --header "Content-Type: application/json" --request GET --data '{"name":"Butterball"}' http://localhost:5000/cat/get

You will get back the JSON of the cat object.

{ "breed": "Tabby", "color": "Orange", "name": "Butterball" }

If you give the API the name of a non existant cat, it will tell you.

curl --header "Content-Type: application/json" --request GET --data '{"name":"Fake Cat Name"}' http://localhost:5000/cat/get

Gives:

{ "message": "No cat by that name found!"}

This matches the behavior outlined in the docs.

For more about how it works, please consult the full dos documentation.
